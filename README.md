#  Day 4 – Cyber Attacks & Detection Fundamentals  


---

## Objective  
- Understand common cyber attack techniques  
- Identify key indicators of attacks  
- Learn SOC detection fundamentals  
- Apply basic log and behavior analysis  

---

## Topics Covered  

### Phishing  
- Social engineering attack  
- Uses fake emails, urgent language, and malicious links  
- Targets sensitive data (credentials, OTPs)  

---

### Malware  
- Malicious software affecting systems  

**Types:**  
- Virus  
- Ransomware  
- Spyware  

---

### Brute Force  
- Multiple password attempts to gain access  
- Exploits weak authentication  

---

## SOC Detection  

- Log monitoring  
- Pattern recognition (failed logins, unusual activity)  
- Identifying threat indicators (phishing, malware)  

---

## SOC Workflow  

Logs → Collection (SIEM) → Alerts → Investigation → Response  

---

##  Activity  

**Scenario:**  
> "Your account will be suspended. Click here immediately to verify your details."

**Result:**  
Phishing attempt (urgent language + suspicious link + no trusted source)

---

## Assignment  

**Example:** Fake PayPal email  

**Reasons:**  
- Fake domain  
- Urgent message  
- Requests sensitive data  

**Prevention:**  
- Verify URLs  
- Avoid unknown links  
- Enable MFA  

---

##  Key Takeaway  

Cyber attacks often rely on user manipulation.  
Detection begins with awareness, log analysis, and identifying anomalies.
